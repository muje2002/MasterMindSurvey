# PRD: MasterMind MVP 검증 설문조사 웹앱

## 1. 프로젝트 개요

- **프로젝트명:** MasterMind Survey
- **목적:** 리더 전용 프라이빗 AI 비서 "MasterMind" MVP 검증을 위한 프리미엄 웹 설문조사 시스템
- **타겟 사용자:** C-Level 임원, 사업부 리더, 스타트업 대표 등 경영진
- **배포 방식:** GitHub Pages 또는 Vercel (무료 호스팅)
- **데이터 저장:** Google Sheets API (무료, 실시간 자동 기록)
- **QR 코드:** 설문 URL 기반 자동 생성 (인쇄/공유용)

---

## 2. 핵심 요구사항

### 2.1 디자인 & UX
- **다크 프리미엄 UI**: 리더급 분위기에 맞는 고급스러운 다크 톤 디자인
- **컬러 팔레트**: 다크 배경(#0A0A0F ~ #1A1A2E), 골드/앰버 액센트(#D4AF37, #F5C842), 화이트 텍스트
- **타이포그래피**: 세리프 + 산세리프 조합 (제목: Playfair Display 또는 유사, 본문: Inter/Pretendard)
- **반응형**: 모바일 우선 (QR 스캔 → 스마트폰에서 응답)
- **애니메이션**: 부드러운 페이드인/슬라이드 전환, 미니멀한 마이크로 인터랙션

### 2.2 설문 구조 (4파트, 총 11문항)
- **Part 1 (Q1~Q3):** 리더의 고독 및 기존 대안 검증 — 객관식 2개 + 주관식 1개
- **Part 2 (Q4~Q6):** AI 사용 경험 및 보안 리스크 검증 — 객관식 3개
- **Part 3 (Q7~Q10):** 지불 의사 및 가격 민감도 (Van Westendorp PSM) — 주관식(금액) 4개
- **Part 4 (Q11):** 확약 도출 — 이메일 입력 1개

### 2.3 진행 UI
- **프로그레스 바**: 상단 고정, Part 1~4 단계 표시 (시각적 진행률)
- **파트별 전환**: 한 파트 완료 → 다음 파트로 슬라이드 전환
- **한 문항씩 표시** 또는 **파트 단위 표시** (파트 단위 추천)
- **뒤로가기**: 이전 파트로 돌아갈 수 있음

## 3. 상세 문항 정의

### Part 1: 리더의 고독 및 기존 대안 검증

**Q1 (객관식 — 단일 선택)**
> 가장 최근 중대한 전략적 의사결정을 내리실 때, 사내 임원진이나 실무자들과 모든 정보를 투명하게 상의하기 어려웠던 경험이 있으십니까?

선택지:
- 매우 자주 있다
- 가끔 있다
- 거의 없다
- 전혀 없다

**Q2 (주관식 — 단답형)**
> 위 문항에서 상의가 어려웠다면, 주로 어떤 외부 채널(예: 고가의 경영 컨설팅, 프라이빗 조찬 모임, 지인 네트워킹 등)을 통해 조언을 구하셨습니까?

입력: 텍스트 입력란 (placeholder: "예: 외부 컨설팅, 동종업계 지인 모임 등")

**Q3 (객관식 — 단일 선택)**
> 위에서 언급하신 외부 조언이나 프리미엄 네트워킹을 얻기 위해 연간 회사 차원(법인카드 등)에서 지출하는 비용은 대략 어느 정도입니까?

선택지:
- 100만 원 미만
- 100만 원 ~ 300만 원
- 300만 원 ~ 500만 원
- 500만 원 이상

### Part 2: AI 사용 경험 및 보안 리스크 검증

**Q4 (객관식 — 단일 선택)**
> 최근 3개월 내에 범용 생성형 AI(ChatGPT, Claude 등)를 업무에 활용해 보신 적이 있습니까?

선택지:
- 매일 사용한다
- 주 1~2회 사용한다
- 한두 번 시도해 보았다
- 사용해 본 적 없다

**Q5 (객관식 — 단일 선택)**
> 기존 AI 서비스를 사용하실 때, 회사의 민감한 재무 데이터나 전략 기밀이 서버에 저장되거나 유출될 것이 우려되어 데이터 입력을 포기하신 경험이 있습니까?

선택지:
- 여러 번 있다
- 한두 번 있다
- 없다

**Q6 (객관식 — 단일 선택)**
> 기존 AI의 답변이 너무 길고 장황하여, 바쁜 업무 중 즉각적인 의사결정에 오히려 방해가 된다고 느끼신 적이 있습니까?

선택지:
- 매우 동의한다
- 약간 동의한다
- 동의하지 않는다

### Part 3: 지불 의사 및 가격 민감도 (Van Westendorp PSM)

파트 안내 문구:
> 다음은 '입력 즉시 데이터가 완벽히 파기되는(Zero-Retention) 임원 전용 맞춤형 AI 섀도우 캐비닛(Shadow Cabinet)' 서비스의 구독료에 대한 질문입니다. 법인카드 결제 기준, 적절하다고 생각하시는 월 구독료를 숫자로 적어주세요.

**Q7 (주관식 — 금액 입력)**
> 이 서비스가 '너무 비싸서' 구매를 전혀 고려하지 않을 월 구독료는 얼마입니까?

입력: 숫자 입력 (단위: 만 원), placeholder: "예: 100"

**Q8 (주관식 — 금액 입력)**
> '비싸다'고 생각하지만, 여전히 구매를 고려해 볼 만한 월 구독료는 얼마입니까?

입력: 숫자 입력 (단위: 만 원), placeholder: "예: 50"

**Q9 (주관식 — 금액 입력)**
> 비용 대비 가치가 뛰어나 '가성비가 매우 좋다(Bargain)'고 느껴지는 월 구독료는 얼마입니까?

입력: 숫자 입력 (단위: 만 원), placeholder: "예: 20"

**Q10 (주관식 — 금액 입력)**
> 너무 저렴해서 서비스의 수준이나 '데이터 보안의 신뢰성이 의심되는' 월 구독료는 얼마입니까?

입력: 숫자 입력 (단위: 만 원), placeholder: "예: 5"

### Part 4: 확약(Commitment) 도출

**Q11 (이메일 입력)**
> 본 설문 결과에 기반하여, 대화 내용이 100% 저장되지 않는 리더 전용 프라이빗 AI의 클로즈드 베타 버전이 출시될 예정입니다. 가장 먼저 초대장(Invite)과 본 벤치마크 조사 결과 리포트를 받아보시겠습니까? 참여를 원하시면 명함상의 이메일 주소를 남겨주세요.

입력: 이메일 입력란 (선택사항, 필수 아님)

## 4. 기술 스택

| 항목 | 기술 | 비고 |
|------|------|------|
| 프론트엔드 | Vanilla HTML/CSS/JS (단일 페이지) | 프레임워크 불필요, 빠른 로딩 |
| 스타일링 | CSS Custom Properties + 애니메이션 | 다크 프리미엄 테마 |
| 폰트 | Google Fonts (Playfair Display + Pretendard/Inter) | 한/영 모두 지원 |
| 문항 관리 | Google Sheets "문항" 시트 → JS fetch | 비개발자도 문항 수정 가능 |
| 데이터 저장 | Google Apps Script Web App → Google Sheets "응답" 시트 | 무료, CORS 우회 |
| QR 코드 | qrcode.js 라이브러리 또는 별도 생성 스크립트 | 설문 URL → QR 이미지 |
| 호스팅 | GitHub Pages 또는 Vercel | 무료, HTTPS 자동 |
| 도메인 | (선택) 커스텀 도메인 연결 가능 | survey.mastermind.ai 등 |

## 5. 데이터 흐름

### 5.1 설문 문항 로드 (동적)
```
[설문 페이지 로드]
       ↓
[JS → Google Apps Script GET 요청]
       ↓
[Apps Script → "문항" 시트 읽기 → JSON 반환]
       ↓
[JS가 JSON 기반으로 설문 UI 동적 렌더링]
```

### 5.2 응답 제출
```
[사용자 QR 스캔] → [설문 웹페이지 로드] → [문항 데이터 fetch]
       ↓
[Part 1~4 응답 작성]
       ↓
[제출 버튼 클릭]
       ↓
[JavaScript → Google Apps Script Web App (POST)]
       ↓
[Google Apps Script → "응답" 시트에 행 추가]
       ↓
[완료 화면 + 감사 메시지 표시]
```

### 5.3 Google Sheets 구조 (시트 2개)

#### 시트 1: "문항" (관리자가 편집)
| 컬럼 | 내용 | 예시 |
|------|------|------|
| A | 문항 ID | Q1 |
| B | 파트 번호 | 1 |
| C | 문항 유형 | radio / text / number / email |
| D | 문항 텍스트 | 가장 최근 중대한 전략적 의사결정을... |
| E | 선택지 (쉼표 구분) | 매우 자주 있다,가끔 있다,거의 없다,전혀 없다 |
| F | 필수 여부 | Y / N |
| G | placeholder | 예: 외부 컨설팅, 동종업계 지인 모임 등 |
| H | 파트 제목 | 리더의 고독 및 기존 대안 검증 |
| I | 파트 안내 문구 | (Part 3 등에서 사용하는 긴 안내 텍스트) |
| J | 단위 | 만 원 (금액 입력 시 표시) |

#### 시트 2: "응답" (자동 기록)
| 컬럼 | 내용 |
|------|------|
| A | 타임스탬프 (자동) |
| B~ | 각 문항 ID에 매핑된 응답값 (동적) |

> **핵심:** 문항 시트의 행을 추가/삭제/수정하면 설문 UI가 자동으로 변경됨. 코드 수정 및 재배포 불필요.

## 7. QR 코드 생성

- 설문 페이지 URL을 QR 코드로 변환
- QR 이미지 출력: PNG (고해상도 300+ DPI)
- 용도: 명함 크기 카드 인쇄, 프레젠테이션 슬라이드 삽입
- QR 중앙에 MasterMind 로고 삽입 (선택)
- 별도 `qr.html` 페이지에서 QR 코드 생성 및 다운로드 기능 제공

---

## 8. 유효성 검증

- 객관식: 선택하지 않으면 다음 파트 진행 불가
- 금액 입력 (Q7~Q10): 숫자만 입력 가능, 빈 값 허용하지 않음
- 이메일 (Q11): 형식 검증 (선택사항이므로 빈 값 허용)
- 제출 중복 방지: 제출 버튼 1회만 클릭 가능 (로딩 스피너)

---

## 9. 비기능 요구사항

- **로딩 속도**: 첫 로드 2초 이내 (CDN 폰트 + 경량 JS)
- **접근성**: 모바일 터치 친화적 (버튼 최소 44px)
- **오프라인 대비**: 네트워크 에러 시 재시도 안내
- **개인정보**: 이메일 외 개인식별정보 수집하지 않음
- **브라우저 지원**: Chrome, Safari, Samsung Internet (최신 2버전)

## 11. Google Apps Script 설정 가이드

1. Google Sheets 새 문서 생성
2. 확장 프로그램 → Apps Script 열기
3. `Code.gs` 코드 붙여넣기
4. 배포 → 새 배포 → 웹 앱 → "누구나 액세스 가능" 설정
5. 배포 URL을 `submit.js`의 `SCRIPT_URL` 변수에 입력
